# 동서양 퓨전 타로 78카드 덱

Korean Myth + East/West Esoteric Tarot + Twelve Zodiac (십이지신) + Four Guardians (사신도)

---

## 1. 덱 컨셉 개요

- 목표: 전통 타로 구조(22 Major + 56 Minor)를 한국/동아시아 신화 · 설화 · 상징체계와 융합.
- 핵심 축:
  - 메이저 아르카나: 영적 여정(순수 → 통합)
  - 마이너 아르카나: 사방신(주작/현무/청룡/백호) + 십이지신 코트 캐릭터
  - 코트 카드: Page / Knight / Queen / King = (삼합 기반 3 동물 + 수호신 Guardian Variant)
- Guardian Variant: 같은 랭크(주로 King 또는 Queen)의 ‘신적 아바타’ 대체 일러스트(선택적).

## [카드 프레임 일관성 전략 및 Midjourney -cref 활용 가이드]

### 1. 프롬프트만으로 카드 프레임(색상/무늬/테두리) 일관성 유지 가능성

- Midjourney는 프롬프트에 "ornate gold black frame", "consistent tarot card style", "unified color palette" 등 반복적으로 명시하면 어느 정도 프레임 일관성을 유지할 수 있음.
- 하지만 프롬프트만으로 완벽하게 동일한 프레임/테두리/무늬를 보장하긴 어렵고, 카드별로 미묘한 차이가 발생할 수 있음.
- 특히 색상·무늬의 세부 디테일(예: 특정 문양, 두께, 질감 등)은 프롬프트만으로 완벽하게 통제 불가.

### 2. -cref(Reference Image) 전략

- Midjourney의 `--cref`(Custom Reference) 기능은 **참조 이미지의 스타일, 색상, 프레임, 무늬**를 강하게 반영함.
- 카드 프레임/테두리/무늬/색상 등 "외곽 통일성"이 중요한 경우, **프레임만 그려진 샘플 이미지를 cref로 지정**하고, 각 카드 프롬프트에 `--cref <프레임이미지URL>`을 추가하면, 다양한 카드 컨텐츠(인물, 동물, 상징 등)와 무관하게 프레임/색상/무늬가 일관되게 적용됨.
- 이 방식은 카드별로 다양한 주제와 색감이 들어가도, **프레임/외곽/무늬는 cref 이미지와 거의 동일하게 유지**됨.

### 3. 추천 Midjourney 워크플로

1. **프레임 전용 샘플 이미지 제작**

   - 원하는 카드 프레임(색상, 무늬, 테두리, 각종 장식 등)만 그린 이미지를 먼저 생성.
   - 이 이미지를 Midjourney에 업로드하여 URL 확보.

2. **각 카드 프롬프트에 --cref 적용**

   - 카드별 프롬프트 예시:
     ```
     tarot card, The Fool, young Korean wanderer, ... (카드별 컨텐츠), --ar 2:3 --s 750 --style raw --cref <프레임이미지URL>
     ```
   - 이렇게 하면 카드별로 다양한 인물/동물/상징이 들어가도, 프레임/외곽/무늬/색상은 cref 이미지와 동일하게 유지됨.

3. **프롬프트 내 프레임 관련 키워드는 최소화**
   - "ornate gold black frame" 등 프레임 관련 키워드는 빼고, 카드별 컨텐츠와 스타일만 기술.
   - 프레임 통일은 cref가 담당, 컨텐츠 다양성은 프롬프트가 담당.

### 4. 결론 및 전략 요약

- **프롬프트만으로는 프레임/무늬/색상 완전 통일은 한계**가 있음(유사성은 가능, 완전 동일은 어려움).
- **Midjourney --cref 기능을 활용해 프레임 샘플 이미지를 참조**하면, 카드별 컨텐츠와 무관하게 프레임/색상/무늬의 일관성 보장 가능.
- 추천 전략:
  1. 프레임만 그린 샘플 이미지를 제작·업로드
  2. 카드별 프롬프트에 `--cref <프레임이미지URL>` 추가
  3. 카드별 컨텐츠는 자유롭게 프롬프트로 기술

---

**실전 예시**

```
tarot card, The Sun, Korean children sunflower field smiling sun, cyberpunk mystical vibe --ar 2:3 --s 750 --style raw --cref https://.../your-frame-image.png
```

- 위와 같이 하면 카드별로 다양한 컨텐츠를 구현하면서도, 프레임/무늬/색상 테마는 완벽히 통일 가능.
